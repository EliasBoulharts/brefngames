Description

There are two different possibilities to implement the parameter concept.

First, the concrete classes (ColorParameter, GridParameter, etc) extend AbstractParameter. In this case the attribute "value" of AbstractParameter needs to be set with the correct type. (For example, ColorParameter has to set value to Color type).

Second, the concrete classes implement Configurable. For this possibility, when getValue() will be called, the real type will be returned.
